# CryptX-UI

CryptX-UI is a responsive cryptocurrency dashboard I built using React, Vite, and Tailwind CSS. It provides a clean and intuitive interface for exploring cryptocurrency data through interactive charts, key market metrics, and a modern user experience.

## Features

- Dashboard with cryptocurrency metric cards
- Interactive price charts
- Search functionality
- Fully responsive layout
- Modern UI built with Tailwind CSS
- Fast development with Vite

## Technologies Used

- React
- Vite
- Tailwind CSS
- Recharts
- JavaScript (ES6+)

## Installation

Clone the repository:

```bash
git clone https://github.com/thabangsonique/CryptX-UI.git
```

Navigate to the project:

```bash
cd CryptX-UI
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
